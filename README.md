# 후기
아쉬움이 많이 남습니다. 기존 개발하던 방식과 많이 달라 시간 소모의 요인이 된 것 같습니다.
과제를 진행하는 중 다시한번 개발의 재미를 느낄수 있어 좋은 시간이 된것 같습니다.
다음번에 같은 방식의 세미프로젝트를 과제로 한다면 더 좋은 결과가 있지 않을까 생각합니다.
이 기회를 발판삼아 더 나아진 모습을 보여드리겠습니다. 감사합니다.
